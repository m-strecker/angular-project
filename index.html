Reasons to Consider Using Angular:

  1. Supported by Google
  2. TypeScript Usage
  3. Declarative User Interface
  4. Plain Old JavaScript Objects
  5. Progressive Web Application and Single Page Application Functionality
  6. Simplified Model-View-Controller Architecture
  7. Modular Structure
  8. Code Consistency and Easy Testing

Angular 7 Improvements and Features include the following:

  1. CLI Prompts were updated with new features to help users discover built-in SCSS support, routing and more.
  2. Angular Material & Component Dev Kit (CDK) were updated to include refresh, virtual scrollling, large data lists, and dynamic loading functionality.
  3. A Drag & Drop Module was added.
  4. Virutal Scrolling to load items from the DOM depending upon visible parts for faster experiences with large lists.
  5. Peformance Improvements
  6. Documentation Updates
  7. Dependency Updates for Node, TypeScript and RxJS
